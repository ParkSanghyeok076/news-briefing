# 📋 종합뉴스 브리핑 지침

## 입력
`python3 scripts/fetch_news.py --source general` 로 수집한 JSON 객체

## 출력 목표
- 전체 **15~20개 항목** (중복 제거 후)
- 한국어 마크다운 보고서
- 저장: `reports/YYYY-MM-DD/general_HHMMSS.md`

---

## 섹션 구조

### 1. 🤖 AI 동향 (5~8개)
- 소스: ai_news, miilk_ai, aitimes_kr, aitimes_com, ai_magazine

### 2. 💻 IT·테크 (5~7개)
- 소스: geek_news, mk_ai, tech_review

### 3. 💼 비즈니스·경영 (3~5개)
- 소스: ttimes, mk_ai (경영/비즈니스 관련 기사)

---

## 항목 형식 (4줄 고정)

```
**[제목](링크)**
출처: {source_name} | 시간: {time} | 언어: {한국어/영어}

요약: {기사 핵심 내용 2~3문장, 한국어로}
인사이트: {왜 지금 중요한지 한 줄}
```

---

## 보고서 헤더 형식

```markdown
# 📰 종합뉴스 브리핑
**{YYYY년 MM월 DD일}** | 수집 소스: 9개 | 총 {N}건

---
```

---

## 규칙

- **언어**: 영어 기사 제목·내용 모두 한국어로 번역
- **중복 금지**: 동일 주제 기사는 가장 상세한 것 1개만 포함
- **데이터 진실성**: 수집된 JSON 항목만 사용, 임의 생성 금지
- **결과 부족**: 전체 5개 미만이면 ⚠️ 표시 후 소스 확대 재시도
- **시간 필드**: 수집 시간 또는 기사 발행 시간 필수 포함
- **영어 원문 링크**: URL은 원문 그대로 유지 (번역하지 않음)
